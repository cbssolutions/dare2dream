POS Disable Options

The "POS Disable Options" is a module for Odoo Point of Sale that allows system administrators to enable or disable specific functions and buttons in the POS user interface on a per Point of Sale basis. This module enhances security and control over the Point of Sale by limiting the actions that employees can perform.

Features
  • Enable / Disable Payments
  • Enable / Disable Quantity Adjustment
  • Enable / Disable Removing Order Lines
  • Enable / Disable Decreasing Quantity
  • Enable / Disable Discounts
  • Enable / Disable Price Editing
  • Enable / Disable Manual Customer Selection
